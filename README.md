# Small Description and why I did this project

This small project shows the mechanics of Deep Learning under the hood. The goal was to understand the math behind backpropagation. 
It helped me understand the logic and math behind backward functions and ReLU. 

Standard Deep Learning frameworks like PyTorch and TensorFlow abstract away the mathematical complexity of neural network training. I built this engine to gain a deep, first-principle understanding of:
- Computation Graphs
- Backpropagation
- Topological Sorting

# Features

- Core Operations: Addition, multiplication, and subtraction with automatic gradient tracking.
- Non-Linearity: ReLU activation function to enable complex pattern learning.
- Gradient Accumulation: Proper handling of multi-step gradients using the chain rule.

I intentionally avoided high level AI frameworks and focused on understanding the core logic
